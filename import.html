Connection status

Start with YouTube, then Last.fm.

YouTube uses Google OAuth in the browser. Last.fm can import public listening history with a free API key. CSV and paste remain the safest fallback.

To authorize YouTube / YouTube Music
  1. Create a Google Cloud project.
  2. Enable YouTube Data API v3.
  3. Create an OAuth Client ID for a web app.
  4. Add https://johnny0103.github.io as an authorized JavaScript origin.
  5. Copy the Client ID into the Moodix YouTube config.
To authorize Last.fm
  1. Create a free Last.fm API account.
  2. Copy the API key into the Moodix Last.fm config.
  3. Users can import from public usernames without typing full playlists.
Spotify later
  1. Create a Spotify Developer app.
  2. Add the redirect URI shown on this page.
  3. Copy the Client ID into the Moodix Spotify config.
