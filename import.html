Connection status

Start with YouTube, then Last.fm.

YouTube uses Google OAuth in the browser. Last.fm uses one free developer API key plus a user’s public username. CSV and paste remain the safest fallback.

To authorize YouTube / YouTube Music
  1. YouTube Data API v3 is the direct account import path.
  2. The Moodix OAuth Client ID is now configured in app.js.
  3. Keep https://johnny0103.github.io as an authorized JavaScript origin in Google Cloud.
  4. Add your own Google account under OAuth test users while the app is unverified.
  5. Make sure that Google account has opened YouTube and has an active YouTube channel/profile.
  6. After verification, users can connect normally from this page.
To authorize Last.fm
  1. Go to last.fm/api/account/create.
  2. Create an API account and copy the API key, not the shared secret.
  3. The Moodix Last.fm key is now configured in app.js.
  4. Users can import by entering only their public Last.fm username.
Spotify later
  1. Create a Spotify Developer app.
  2. Add the redirect URI shown on this page.
  3. Copy the Client ID into the Moodix Spotify config.
