Music import

How do you want to bring music in?

Choose one path. Moodix will only show the fields you need.

YouTube

Connect and choose a playlist.

YouTube is configured. Connect your Google account, choose a playlist, and Moodix will import the track titles.

Google setup already used: Authorized JavaScript origin: https://johnny0103.github.io If Google says the app is unverified, add your email as a test user or use Advanced while testing. If Moodix says channel not found, open YouTube once with this Google account and create/activate a channel.

Last.fm

Import by username.

Enter a Last.fm username to import public recent, top, or loved tracks once an API key is configured.

Manual import

Add a few songs fast.

Type a song and artist, add it to the list, then save. Moodix can still rank the music by your mood.

Paste or upload a list instead

Choose YouTube, Last.fm, or Manual import to start.

Simple guide

Pick the easiest path.

YouTube is best if the user has playlists. Last.fm is best if the user already tracks listening history. Manual import is best for a quick prototype test.

YouTube
  1. Connect Google.
  2. Choose a playlist.
  3. Moodix imports song titles.
Last.fm
  1. Enter username.
  2. Choose recent, top, or loved tracks.
  3. No playlist permission needed.
Manual import
  1. Add songs one by one.
  2. Or paste/upload a list.
  3. Save and continue to results.
